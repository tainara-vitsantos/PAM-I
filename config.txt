-- Passo a passo para a criação de um app --
----------------------------------------------------

1. Abra o Windows PowerShell;

2. Navegue até a pasta onde deseja criar o projeto;

3. Execute o comando:
npx create-expo-app nomeDoApp --template blank

4. Acesse a pasta do app que foi criado, com o comando:
cd nomeDoApp

5. Abra o Visual Studio Code digitanto o comando: 
code .

6. Abra o app no navegador com o comando:
npm run web 
ou
npm run android

npm install
